# RTL Support

ActionMenu has built-in support for right-to-left (RTL) languages like Arabic, Hebrew, and Persian.

## Basic Usage

Set the `dir` prop to enable RTL mode:

```tsx
<ActionMenu.Surface
  dir="rtl"
  menu={menu}
/>
```

## Automatic Direction Detection

Detect direction from HTML or context:

```tsx
// Option 1: Inherit from HTML
<html dir="rtl">
  <ActionMenu.Surface menu={menu} />
  {/* Automatically uses RTL */}
</html>

// Option 2: Use React context
import { DirectionProvider } from '@radix-ui/react-direction'

<DirectionProvider dir="rtl">
  <ActionMenu.Surface menu={menu} />
</DirectionProvider>
```

## Affected Behaviors

### Layout

RTL mode flips the entire layout:

- Menus open to the left instead of right
- Submenus appear on the left side
- Scrollbars appear on the left
- Text aligns to the right

### Icons

Position icons appropriately:

```tsx

  label: 'Next',
  icon: dir === 'rtl' ? <ChevronLeft /> : <ChevronRight />

```

Or use logical properties:

```css
.icon {
  margin-inline-start: 8px; /* Adapts to direction */
}
```

## Styling for RTL

### Logical Properties

Use CSS logical properties for automatic adaptation:

```css
/* Instead of margin-left/right */
.item {
  margin-inline-start: 8px;
  margin-inline-end: 16px;
  padding-inline: 12px;
}

/* Instead of border-left/right */
.separator {
  border-inline-start: 1px solid;
}

/* Instead of text-align: left/right */
.text {
  text-align: start;
}
```

### Direction-Specific Styles

Apply styles based on direction:

```tsx
const theme = {
  classNames: {
    item: cn(
      'px-4 py-2',
      dir === 'rtl' ? 'text-right' : 'text-left'
    )
  }
}
```

Or use CSS:

```css
[dir="rtl"] .menu-item {
  text-align: right;
}

[dir="ltr"] .menu-item {
  text-align: left;
}
```

## Internationalization

Combine with i18n libraries:

```tsx
import { useTranslation } from 'react-i18next'

function MyMenu() {
  const { t, i18n } = useTranslation()
  const dir = i18n.dir()

  return (
    <ActionMenu.Surface
      dir={dir}
      menu={{
        title: t('menu.title'),
        nodes: [
          { label: t('menu.profile') },
          { label: t('menu.settings') }
        ]
      }}
    />
  )
}
```
