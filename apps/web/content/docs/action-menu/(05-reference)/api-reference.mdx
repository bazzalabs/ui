## TypeScript Configuration

The action menu is built with strict TypeScript. Ensure your `tsconfig.json` has:

```json
{
  "compilerOptions": {
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Generic Type Parameters

Many types accept generic parameters for custom data:

```tsx
type CustomData = {
  priority: 'low' | 'high'
  category: string
}

// Use with MenuDef
const menu: MenuDef<CustomData> = {
  nodes: [
    {
      kind: 'item',
      id: 'task',
      label: 'Task',
      data: {
        priority: 'high',
        category: 'work',
      },
    },
  ],
}
```
